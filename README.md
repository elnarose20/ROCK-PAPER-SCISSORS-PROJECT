# ROCK-PAPER-SCISSORS-PROJECT
ROCK PAPER SCISSORS THE ROCK PAPER SCISSORS PROJECT IS A SIMPLE INTERACTIVE WEB APPLICATION DEVELOPED USING PYTHON AND STREAMLIT . THE GAME ALLOWS USES TO CHOOSE BETWEEN ROCK,PAPER,SCISSORS,WHILE THE COMPUTER RANDOMLY SELECTS ITS CHOICE.BASED ON THE GAMERULES,THE PROGRAM DETERMINES WHETHER THE USER WINS, LOSES, OR TIES.
